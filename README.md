# Telegram-бот для продажи билетов

Простой стартовый проект на Python, aiogram 3.x, PostgreSQL и SQLAlchemy async.

## Запуск

```bash
cp .env.example .env
# заполните BOT_TOKEN, MANAGERS_CHAT_ID, MANAGER_IDS

docker compose up --build
```

## Как пользоваться

### Клиент

1. `/start`
2. Отправить телефон через кнопку Telegram.
3. Выбрать действие:
   - `Купить билет`
   - `Связаться с менеджером`
   - `Мои заказы`

### Менеджер

1. Указать свой Telegram ID в `MANAGER_IDS`.
2. Перезапустить бота.
3. Написать боту `/manager`.
4. Управлять мероприятиями через кнопки:
   - `Создать мероприятие`
   - `Список мероприятий`
   - `Редактировать мероприятие`
   - `Удалить мероприятие`

### Группа менеджеров

1. Добавьте бота в группу.
2. Укажите ID группы в `MANAGERS_CHAT_ID`.
3. Новые заказы будут приходить в эту группу с inline-кнопками статусов.

## Примечания

- В стартовой версии таблицы создаются через `Base.metadata.create_all()`.
- Для production лучше добавить Alembic и перейти на миграции.
- Удаление мероприятия запрещено, если по нему уже есть заказы. Для production лучше добавить soft delete через поле `is_active`.
